# Elevance Health AI Assistant

A conversational RAG application that answers questions about 
patient records, insurance coverage, and claims processing.

## Features
- Document retrieval using semantic search
- Conversation memory
- Clean web interface

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Add your Anthropic API key to .env
3. Run: python app.py
4. Visit: http://127.0.0.1:5000
